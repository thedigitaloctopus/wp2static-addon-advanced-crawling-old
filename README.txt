Adds advanced URL detection options for WP2Static.

== Description ==

Have more control over the URL detection of WP2Static.


== Installation ==

Upload this plugin's ZIP to your WordPress plugins page within your dashboard.

Activate the plugin, then navigate to your WP2Static main options page to see
 the new detection options available.
